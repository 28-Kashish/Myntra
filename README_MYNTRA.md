# Myntra Sales Analytics Dashboard

## 🎯 Project Overview
An end-to-end data analytics project simulating Myntra's e-commerce sales data with data generation, cleaning, database storage, and interactive visualization.

## 📊 Features

### Data Generation
- 2500+ records with realistic e-commerce patterns
- Intentional data quality issues (nulls, duplicates, outliers)
- Product categories, brands, sizes, colors, pricing
- Customer demographics and behavioral data

### Data Cleaning
- Duplicate removal
- Missing value handling
- Data type correction
- Outlier treatment
- Derived feature engineering

### Database Storage
- MySQL database with optimized schema
- Indexes for query performance
- Database views for analytics

### Dashboard Visualizations
- **Sales Analysis**: Revenue trends, monthly performance, seasonal patterns
- **Product Analytics**: Category performance, brand analysis, sub-category insights
- **Rating Analysis**: Rating distribution, category-wise ratings, review patterns
- **Delivery Analytics**: Delivery status, performance metrics, delivery days
- **Customer Insights**: Customer segments, city analysis, loyalty program impact
- **Pricing Analysis**: Discount patterns, price categories, coupon effectiveness
